# Student Attendance Dashboard

## Intern Alpha Internship — Power BI Task 1

### Project Overview
An interactive student attendance dashboard built using Google Looker Studio (alternative to Power BI). This dashboard provides comprehensive insights into student attendance patterns with visual analytics and filtering capabilities.

### Features
- Monthly attendance % by class
- Pie chart for attendance distribution
- Line chart for 12 month trends
- Student level data table
- Filters for Class, Section and Month

### Tools Used
- Google Looker Studio
- Google Sheets
- CSV Data (240 rows, 20 students, 12 months)
